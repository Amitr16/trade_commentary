DTCC Trade Commentary Dashboard

About This Application

This web application provides AI-generated commentary on DTCC trade data. The system consists of:

  • DTCC Fetcher: Runs every minute to collect trade data and update dtcc_trades.csv
  • Commentary Generator: Creates AI-powered analysis using OpenAI GPT
  • Web Interface: Displays the commentary with real-time refresh capability
